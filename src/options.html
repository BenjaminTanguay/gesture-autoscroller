Getting Started

Gesture AutoScroller provides hands-free reading through simple gestures. The extension only works on sites you've added to the whitelist.

Quick Tip: 3-Finger Tap Mobile Tap with 3 fingers anywhere on the page to quickly enable/disable the extension for the current site. This is the fastest way to add sites to your whitelist!

Activation Gestures

Mobile Mobile

Two-Finger Tap Tap simultaneously with two fingers anywhere on the page to activate autoscroll.
Three-Finger Tap Tap with three fingers to add/remove the current site from your whitelist. A toast message will confirm the action.
Four-Finger Tap Tap with four fingers to activate the element picker, which lets you select the "Next Page" button for auto-navigation. The site must be whitelisted first.

Desktop Desktop

Scroll Down Simply scroll down with your mouse wheel or trackpad to activate autoscroll. Works from any position on the page.

Autoscroll Controls

Speed Adjustment

  • Swipe Up: Increase scrolling speed (works continuously - keep swiping for faster speed)
  • Swipe Down: Decrease scrolling speed (works continuously)

Speed adjustments work in both SCROLLING and PAUSED states. Changes take effect immediately.

Pause/Resume

Single Tap Tap anywhere on the page to pause scrolling. Tap again to resume from the same position.

Stop Autoscroll

Side Swipe (Left or Right) Swipe left or right to completely stop and exit autoscroll mode. Works from both SCROLLING and PAUSED states.

Tap Navigation

When autoscroll is NOT active, you can use tap/click gestures for quick page navigation (works on both mobile and desktop).

Basic Usage

By default, the screen is divided into two zones for scrolling:

  • Horizontal Layout (Default): Tap/Click left side = scroll up, tap/click right side = scroll down
  • Vertical Layout: Tap/Click top half = scroll up, tap/click bottom half = scroll down

Note: Tap navigation is disabled when autoscroll is active, and won't interfere with links or buttons.

Customizing Tap Navigation

You can customize tap navigation in Settings > Tap tab to match your reading preferences:

Tap Zone Layout Choose between horizontal split (left/right) or vertical split (top/bottom) based on how you hold your device.
Tap Zone Size Adjust the relative size of scroll-up vs scroll-down zones (10% to 90%).
Tap Scroll Distance Configure how much the page scrolls when you tap, from 10% to 100% of viewport height.

Auto-Navigate to Next Page

The Auto-Navigate feature allows continuous hands-free reading by automatically clicking "Next Page" buttons when autoscroll reaches the bottom of a page.

How It Works

  1. When autoscroll reaches the bottom of a whitelisted page, it pauses for a configurable countdown (default: 3 seconds)
  2. A countdown message appears at the bottom of the screen: "Navigating to next page... (Tap to cancel)"
  3. After the countdown, the extension automatically clicks the configured "Next Page" button
  4. Optionally, autoscroll can resume automatically on the new page

Setting Up the "Next Page" Button

For this feature to work, you need to configure which button/link on each site is the "Next Page" button. There are three methods:

Method 1: Element Picker Button Mobile
  1. Open Settings > Auto-Navigate tab
  2. Select the domain you want to configure
  3. Click "Activate Element Picker" button
  4. Switch to the website tab
  5. Tap on the "Next Page" button or link
  6. The CSS selector will be captured and saved automatically
Method 2: 4-Finger Tap Mobile
  1. Navigate to the whitelisted site in your browser
  2. Tap with 4 fingers anywhere on the page to activate the element picker
  3. Tap on the "Next Page" button or link
  4. The CSS selector will be captured and saved automatically
  5. Return to Settings to verify the selector was saved
Method 3: Right-Click Selection Desktop
  1. Navigate to the whitelisted site in your browser
  2. Right-click on the "Next Page" button or link
  3. Select "Select as 'Next Page' button" from the context menu
  4. The CSS selector will be captured and saved automatically

Testing Your Configuration

After setting up the "Next Page" selector:

  1. Go to Settings > Auto-Navigate tab
  2. Select your configured domain
  3. Click "Test Selector on Current Page" to verify it finds the correct element
  4. A success or error message will appear

Configuration Presets

Presets allow you to save and reuse your configuration settings across multiple domains, making it easy to maintain consistent reading experiences for similar types of content.

Saving a Preset

  1. Configure your desired settings for any domain (tap zones, scroll speeds, auto-navigate settings, etc.)
  2. Go to Settings > Whitelist & Presets tab
  3. Click "Save as Preset" button
  4. Enter a descriptive name (e.g., "Manga Settings", "Novel Settings", "Documentation")
  5. Click "Save Preset"

Loading a Preset

  1. Select the domain you want to apply the preset to from the dropdown at the top
  2. Go to Settings > Whitelist & Presets tab
  3. Click "Load Preset" button
  4. Select the preset you want to apply
  5. All settings (except the domain-specific "Next Button Selector") will be applied immediately

Managing Presets

Click "Manage Presets" to view all saved presets. From here you can:

  • Load: Apply a preset to the currently selected domain
  • Rename: Change the preset name
  • Delete: Remove a preset permanently

What's Included in Presets

Presets save the following settings:

  • Tap Navigation settings (layout, zone sizes, scroll distance)
  • Auto-Scrolling settings (speeds, delays, auto-start)
  • Auto-Navigate settings (countdown delay, auto-start on new page)
  • Advanced settings (text selection, screen wake, auto-stop)

Note: The "Next Button CSS Selector" is domain-specific and not included in presets, as each site has different HTML structure.

Managing the Whitelist

Quick Method Mobile

  1. Navigate to the site you want to add
  2. Tap with 3 fingers anywhere on the page
  3. A toast message confirms the site is now whitelisted
  4. Tap with 3 fingers again to remove it from the whitelist

Settings Page Method

  1. Open the Settings tab (this page)
  2. Scroll to "Active Hosts (Whitelist)"
  3. Type a domain name (e.g., medium.com) and click "Add Host"
  4. Or click "Add current site to whitelist" if you're viewing a page

Troubleshooting

Extension doesn't work on a page Make sure you've added the site to your whitelist. Use the 3-finger tap gesture or add it manually in Settings.
Two-finger tap not working Ensure both fingers touch the screen at the same time. The gesture should be quick (less than 200ms).
Page scrolls when adjusting speed This has been fixed in the latest version. Make sure you're running the most recent version of the extension.
Speed adjustments not working Make sure you're swiping far enough (at least 50 pixels). Swipe slowly and deliberately for better detection.
Tap navigation triggers on buttons/links The extension should automatically detect interactive elements. If you experience issues, please report them.
Desktop scroll activation too sensitive The extension activates on any downward scroll. If this is too sensitive for your use case, you can disable autoscroll in Settings and use the mobile gestures only.
