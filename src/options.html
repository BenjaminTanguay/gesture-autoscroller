Getting Started

Gesture AutoScroller provides hands-free reading through simple gestures. The extension only works on sites you've added to the whitelist.

Quick Tip: 3-Finger Tap Mobile Tap with 3 fingers anywhere on the page to quickly enable/disable the extension for the current site. This is the fastest way to add sites to your whitelist!

Activation Gestures

Mobile Mobile

Two-Finger Tap Tap simultaneously with two fingers anywhere on the page to activate autoscroll.
Three-Finger Tap Tap with three fingers to add/remove the current site from your whitelist. A toast message will confirm the action.

Desktop Desktop

Scroll Down Simply scroll down with your mouse wheel or trackpad to activate autoscroll. Works from any position on the page.

Autoscroll Controls

Speed Adjustment

  • Swipe Up: Increase scrolling speed (works continuously - keep swiping for faster speed)
  • Swipe Down: Decrease scrolling speed (works continuously)

Speed adjustments work in both SCROLLING and PAUSED states. Changes take effect immediately.

Pause/Resume

Single Tap Tap anywhere on the page to pause scrolling. Tap again to resume from the same position.

Stop Autoscroll

Side Swipe (Left or Right) Swipe left or right to completely stop and exit autoscroll mode. Works from both SCROLLING and PAUSED states.

Tap Navigation

When autoscroll is NOT active, you can use tap/click gestures for quick page navigation (works on both mobile and desktop):

  • Tap/Click Left Side: Page up (scroll up one viewport)
  • Tap/Click Right Side: Page down (scroll down one viewport)

Note: Tap navigation is disabled when autoscroll is active, and won't interfere with links or buttons.

Managing the Whitelist

Quick Method Mobile

  1. Navigate to the site you want to add
  2. Tap with 3 fingers anywhere on the page
  3. A toast message confirms the site is now whitelisted
  4. Tap with 3 fingers again to remove it from the whitelist

Settings Page Method

  1. Open the Settings tab (this page)
  2. Scroll to "Active Hosts (Whitelist)"
  3. Type a domain name (e.g., medium.com) and click "Add Host"
  4. Or click "Add current site to whitelist" if you're viewing a page

Troubleshooting

Extension doesn't work on a page Make sure you've added the site to your whitelist. Use the 3-finger tap gesture or add it manually in Settings.
Two-finger tap not working Ensure both fingers touch the screen at the same time. The gesture should be quick (less than 200ms).
Page scrolls when adjusting speed This has been fixed in the latest version. Make sure you're running the most recent version of the extension.
Speed adjustments not working Make sure you're swiping far enough (at least 50 pixels). Swipe slowly and deliberately for better detection.
Tap navigation triggers on buttons/links The extension should automatically detect interactive elements. If you experience issues, please report them.
Desktop scroll activation too sensitive The extension activates on any downward scroll. If this is too sensitive for your use case, you can disable autoscroll in Settings and use the mobile gestures only.
